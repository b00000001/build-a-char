<h1 align='center'>Build-a-Char</h1>

# Description

This is a tool to help facilitate your RPG creation needs. This should take the hassle out of creating a new character by streamlining the character creationg process with boilerplates for each character class, the ability to enter a class name and select from several pre-selected classes.
